# **App Name**: Valence

## Core Features:

- Fun Mode: Experiment with any chemicals. Dangerous reactions trigger an animated blast and a pop-up quiz about the reaction. Users can select the chemical and the quantity.
- Learning Mode: A structured mode for learning about chemical reactions. Students can perform reactions similar to fun mode, but with questions assigned from their uploaded syllabus. Exam mode simulates lab tests.
